# Y Real Estate - Odoo Custom Module

## Description
Real Estate advertising module for Odoo 19 with property management functionality.

## Features
- Property listing management
- Property details (bedrooms, living area, garden, garage)
- Price management (expected and selling price)
- Date availability tracking
- Search and filter capabilities

## Installation
1. Copy module to Odoo addons directory
2. Update app list in Odoo
3. Install "Real Estate Adverting" module

## Module Structure
- `models/`: Business logic and data models
- `views/`: User interface definitions
- `security/`: Access rights and permissions
- `data/`: Initialization and demo data

## Technical Details
- **Model**: `y.realestate.property`
- **Dependencies**: base
- **Version**: 1.0
